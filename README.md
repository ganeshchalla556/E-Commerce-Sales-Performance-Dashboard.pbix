# 📊  E-Commerce-Sales-Performance-Dashboard

## 📌 Overview

This project analyzes retail sales data to track business performance using SQL and Power BI. It provides insights into revenue, profit, and customer trends.

---

## 🎯 Objectives

* Track key business KPIs
* Identify top-performing products
* Analyze regional sales performance
* Understand monthly trends

---

## 🛠 Tools & Technologies

* SQL (PostgreSQL)
* Power BI
* Excel

---

## 📊 Key Metrics

* Total Revenue
* Total Profit
* Profit Margin
* Total Orders
